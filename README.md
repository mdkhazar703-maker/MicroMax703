# MicroMax703
MacroMax703, A QMK-powered macro pad built around a Seeeduino XIAO, with a 3×3 switch matrix and two rotary encoders for volume and screen brightness. It also packs hidden feature unknown to this world.

For Curious People:-
**Hardware**
MCU: Seeeduino XIAO (SAMD21)
Keys: 9× mechanical switches in a 3×3 matrix, one 1N4148 diode per switch
Encoders: 2× rotary encoders with integrated push-button switches (SW10, SW11)

**Software**
Tools Used:-
KICad-PCB desinging;
Onshape- Case desing; 
QMK Firmware- Macroboard Firmware Development;
